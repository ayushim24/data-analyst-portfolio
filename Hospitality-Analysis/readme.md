# Hospitality Analytics Project

## Project Overview
This project focuses on analyzing **Hospitality Industry Data** to understand hotel performance, booking trends, revenue generation, and customer behavior.

The project follows a complete analytics workflow — from **raw data processing to dashboard creation** — using multiple tools for end-to-end analysis.

## Objectives
- Analyze hotel bookings and revenue trends  
- Understand occupancy and performance metrics  
- Identify high-performing hotel types and locations  
- Perform data cleaning and transformation  
- Build interactive dashboards for business insights  

## Project Structure

-  [Excel](./Excel) – Data cleaning and preprocessing  
-  [MySQL](./MySQL) – SQL queries and data analysis  
-  [Power BI](./Power%20BI) – Interactive dashboards  
-  [Tableau](./Tableau) – Data visualizations  
-  [Raw Data](./Raw%20Data) – Original dataset 

## Tools & Technologies Used
- **Excel** – Data cleaning and preparation  
- **MySQL** – Data querying and analysis  
- **Power BI** – Dashboard creation and reporting  
- **Tableau** – Data visualization and storytelling  

## Key Analysis Areas
- Booking Trends Analysis  
- Revenue & Profit Analysis  
- Occupancy Rate Evaluation  
- Customer Behavior Analysis  
- Hotel/Room Type Performance  
- Location-wise Analysis 

## Key Insights
- Revenue varies across different hotel types and locations  
- Occupancy rate impacts overall performance significantly  
- Booking trends change based on time and demand  
- Certain segments contribute more to total revenue  
- Customer behavior patterns help in decision making 

## How to Use
1. Start with **Raw Data** to understand dataset  
2. Use **Excel** for cleaning and preprocessing  
3. Perform analysis using **MySQL**  
4. Explore dashboards in **Power BI** and **Tableau**  
5. Extend analysis based on business requirements
