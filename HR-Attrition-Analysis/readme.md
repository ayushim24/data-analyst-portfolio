# HR Attrition Analytics Project

## Project Overview
This project focuses on analyzing **HR Attrition Data** to understand employee turnover, workforce trends, and key factors affecting attrition.

The analysis is performed using multiple tools including Excel, MySQL, Power BI, and Tableau to provide a complete end-to-end data analytics workflow — from raw data to insights and dashboards.

## Objectives
- Analyze employee attrition and workforce distribution  
- Identify patterns based on salary, department, and experience  
- Perform data cleaning and transformation  
- Build interactive dashboards for insights  
- Apply SQL and visualization tools for business analysis 

## Project Structure

-  [Excel](./Excel) – Data cleaning and preprocessing  
-  [MySQL](./MySQL) – SQL queries and data analysis  
-  [Power BI](./Power%20BI) – Interactive dashboards  
-  [Tableau](./Tableau) – Data visualizations  
-  [Raw Data](./Raw%20Data) – Original dataset

## Tools & Technologies Used
- **Excel** – Data cleaning and preparation  
- **MySQL** – Data querying and analysis  
- **Power BI** – Dashboard creation and reporting  
- **Tableau** – Data visualization and storytelling  

## Key Analysis Areas
- Employee Attrition Analysis  
- Department-wise Performance  
- Salary & Income Distribution  
- Job Role Comparison  
- Work-Life Balance Insights  
- Promotion & Experience Impact

## Key Insights
- Attrition is distributed across departments with slight variation  
- Salary and promotion gap impact employee turnover  
- Work-life balance is relatively consistent across roles  
- Experience level plays a role in attrition trend

## How to Use
1. Start with **Raw Data** for dataset  
2. Use **Excel** for cleaning and preprocessing  
3. Perform analysis using **MySQL**  
4. Explore dashboards in **Power BI** and **Tableau**  
5. Modify and extend analysis as needed
