# HR Attrition Dataset (Raw Data)

## Dataset Overview
This folder contains the **raw HR Attrition dataset** used for analysis in Excel, MySQL, Power BI, and Tableau projects.

The data represents employee information such as department, job role, salary, experience, and attrition status, which is used to analyze workforce trends and employee turnover.

## Purpose of Dataset
- Perform HR analytics and attrition analysis  
- Practice data cleaning and transformation  
- Build dashboards and reports  
- Write SQL queries for insights  
- Learn data visualization techniques 

## Files Included
- **HR_1.csv** → Employee details (department, role, gender, attrition, etc.)  
- **HR_2.csv** → Salary and experience details  

## Dataset Description

### HR_1.csv (Employee Information)
Contains basic employee details:

- EmployeeNumber  
- Department  
- JobRole  
- Gender  
- Attrition (Yes/No)  
- Education  
- HourlyRate  
- JobSatisfaction  
- BusinessTravel  

### HR_2.csv (Salary & Experience Data)
Contains salary and experience-related information:

- Employee ID  
- MonthlyIncome  
- YearsAtCompany  
- TotalWorkingYears 

## Data Preprocessing Notes
- Some columns were cleaned and formatted before analysis  
- EmployeeNumber and Employee ID are used as **primary keys**  
- Data is joined using these keys for analysis  
- Missing or inconsistent values can be handled during processing  

## Data Usage
This dataset is used in the following projects:

-  Power BI Dashboard (HR Attrition Analytics)  
-  Tableau Dashboard  
-  MySQL Analysis  
-  Excel Analysis
