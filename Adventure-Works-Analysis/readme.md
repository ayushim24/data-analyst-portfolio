# Adventure Works Sales Analysis

## Project Overview
This project is a complete **Adventure Works Sales Analysis** solution using **Excel, MySQL, Power BI, and Tableau**. It demonstrates the full data analysis workflow — from raw data handling and SQL querying to interactive dashboard creation.

The goal of this project is to analyze sales performance, identify trends, and generate actionable business insights.

## Objectives
- Analyze overall sales, profit, and orders  
- Identify trends and patterns over time  
- Perform data cleaning and transformation  
- Build interactive dashboards  
- Support data-driven decision-making

## Tools & Technologies Used
Microsoft Excel – Data storage and preprocessing
MySQL – Data querying and analysis
Power BI – Interactive dashboard creation
Tableau – Data visualization and storytelling

## Workflow
Data Collection (Excel)
