# Zomato Sales and Rating Analysis

## Project Overview
This project focuses on analyzing **Zomato restaurant data** to understand sales trends, customer ratings, pricing behavior, and service availability.

The project uses multiple tools to perform **end-to-end data analysis**, starting from raw data cleaning to building interactive dashboards for insights.

## Objectives
- Analyze restaurant distribution across cities and countries  
- Study customer ratings and voting patterns  
- Understand pricing trends (average cost for two)  
- Identify popular cuisines and market demand  
- Evaluate services like online delivery and table booking  
- Build dashboards for business decision-making 

## Tools & Technologies Used
- **MySQL** – Data analysis using SQL queries  
- **Tableau** – Data visualization and storytelling 

## Project Structure 
- [MySQL](./MySQL) – SQL queries and data analysis   
- [Tableau](./Tableau) – Data visualizations

## Key Analysis Areas

### Sales & Pricing Analysis
- Average cost for two across cities and countries  
- Price range distribution (Cheap, Affordable, Expensive, Luxury)  
- Revenue trend indicators using cost data 

### Rating & Customer Insights
- Average restaurant ratings  
- Rating vs votes relationship  
- Highly rated but low-visibility restaurants 

### Cuisine Analysis
- Top cuisines by restaurant count  
- Most preferred cuisines in top cities  
- Cuisine-wise rating comparison 

### Service Analysis
- Restaurants offering online delivery  
- Restaurants offering table booking  
- Comparison of restaurants offering both services 

### Location-Based Insights
- Top cities by restaurant count  
- Country-wise distribution  
- City-wise service availability

## Key Insights
- Majority of restaurants fall in **affordable and cheap categories**  
- High ratings do not always mean high customer engagement (votes)  
- Only a limited number of restaurants provide **table booking services**  
- Online delivery is more common than table booking  
- Certain cities dominate the restaurant market
