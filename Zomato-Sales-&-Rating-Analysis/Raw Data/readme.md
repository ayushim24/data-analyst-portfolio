# Zomato Restaurant Analytics – Raw Data

## Overview
This folder contains the **raw dataset** used for the Zomato Restaurant Analytics project.

The data is in its original form (before cleaning and transformation) and is used for analysis in MySQL and visualization tools like  / Tableau.

## File Details
- **File Name:** `Zomato Dataset.xlsx`
- **Source:** Public dataset (Zomato restaurant data)  
- **Format:** Excel

## Dataset Description
The dataset includes detailed information about restaurants such as:

- Restaurant ID  
- Restaurant Name  
- City  
- Country Code  
- Cuisines  
- Average Cost for Two  
- Rating  
- Votes  
- Has Online Delivery (Yes/No)  
- Has Table Booking (Yes/No)  
- Price Range 

## Data Characteristics
- Contains **thousands of restaurant records**  
- Includes both **categorical and numerical data**  
- May contain:
  - Missing values  
  - Inconsistent formatting  
  - Duplicate entries 

## Preprocessing Steps (Performed Later)
Before analysis, the following steps are applied:

- Data Cleaning  
- Handling missing values  
- Removing duplicates  
- Formatting columns  
- Standardizing categorical values (Yes/No) 

## Purpose of Raw Data
- Serve as the **base dataset** for analysis  
- Used for:
  - SQL queries (MySQL)  
  - Dashboard creation (Tableau)  
- Helps in understanding **real-world messy data handling**
