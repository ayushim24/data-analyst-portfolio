# BlinkIT Grocery Dataset (Raw Data)

## Dataset Overview
This dataset contains **raw grocery sales and outlet data** from BlinkIT.  
It is used for analyzing sales performance, product demand, outlet efficiency, and customer trends.

The data is unprocessed and serves as the base for analysis in Excel, MySQL, Power BI, and Tableau projects.

## Purpose of Dataset
- Perform sales and retail analysis  
- Practice data cleaning and preprocessing  
- Build dashboards and reports  
- Analyze product and outlet performance  
- Generate business insights 

## File Included
- **BlinkIT Grocery Data.xlsx** → Raw dataset containing all records 

## Dataset Description
The dataset includes the following types of information:

### Product Details
- Item Identifier  
- Item Type  
- Item Fat Content  
- Item Visibility  
- Item Weight 

### Sales Information
- Item Outlet Sales  
- Item MRP  
- Sales Performance Metrics 

### Outlet Details
- Outlet Identifier  
- Outlet Type  
- Outlet Size  
- Outlet Location Type (Tier 1, Tier 2, Tier 3)  
- Outlet Establishment Year  

### Customer Metrics
- Customer Rating (if available)  
- Sales Trends based on customer behavior 

## Data Preprocessing Notes
- Dataset may contain missing values (e.g., Item Weight)  
- Some categorical values may need standardization  
- Data types (numeric/text) should be validated before analysis  
- Additional calculated columns can be created for KPIs
