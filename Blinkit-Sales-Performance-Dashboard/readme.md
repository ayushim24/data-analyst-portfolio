# BlinkIT Sales Performance Dashboard

## Project Overview
This project focuses on analyzing **BlinkIT Grocery Sales Data** using **Power BI** to understand sales performance, product demand, and outlet insights.

The workflow includes using **raw data** for analysis and building an interactive dashboard for business decision-making.

## Objectives
- Analyze total sales and revenue trends  
- Identify top-performing products and categories  
- Evaluate outlet performance by type, size, and location  
- Create an interactive dashboard for insights  

## Project Structure

-  [Power BI](./Power%20BI) – Interactive dashboard and report  
-  [Raw Data](./Raw%20Data) – Original dataset  

## Tools & Technologies Used
- **Power BI** – Dashboard creation and visualization  
- **Excel (Raw Data)** – Source dataset  

## Key Analysis Areas
- Sales Performance  
- Product Category Analysis  
- Outlet Performance  
- Location-wise Trends  
- Customer Rating Insights 

## Key Insights
- Top product categories contribute maximum sales  
- Outlet type and size impact overall revenue  
- Tier-based locations show different performance trends  
- Sales distribution varies across outlets  

## How to Use
1. Open the **Raw Data** file  
2. Load the dataset into **Power BI**  
3. Open the `.pbix` file from Power BI folder  
4. Refresh data if required  
5. Use filters and visuals to explore insights
